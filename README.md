# üìö Library Management System (C)

A menu-driven **Library Management System** built in **C**, designed to simulate real-world library operations using structured programming concepts. The system supports **Admin** and **Student** roles with authentication and book issue/return workflows.

## üöÄ Features

### Admin
- Secure login
- Add books to inventory
- Register students
- View issued books report

### Student
- Secure login
- View available books
- Issue a book
- Return a book
